# tests/

Repository-level test assets and cross-cutting suites.

- `fixtures/` — content fixtures with provenance (origin, license, sensitivity, expected behavior). No live PHI.
- `contract/` — cross-language contract conformance.
- `integration/` — multi-component integration.
- `e2e/` — Playwright journeys against the packaged Tauri app.
- `performance/` — performance budgets and benchmarks.
- `security/` — security control verification.

No content fixture without provenance.
