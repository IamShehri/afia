# afia-tauri

Validated Tauri commands, events, and DTO/domain mapping. An adapter layer that
validates schema, size, unlocked-project state, authorization, and rate/concurrency
policy, then delegates to `afia-application`. No business logic, no SQL.

**Depends on:** domain, contracts, application.
